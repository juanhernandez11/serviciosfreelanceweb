# PAGIA-SERVICIOS-FREELANCE

Sitio web profesional de JuanBvtech - Soluciones de Software Architecture

## Estructura del Proyecto

```
PAGIA-SERVICIOS-FREELANCE/
├── src/
│   ├── components/
│   │   ├── Logo.jsx
│   │   ├── Navbar.jsx
│   │   └── MobileMenu.jsx
│   ├── sections/
│   │   ├── Hero.jsx
│   │   ├── Solutions.jsx
│   │   ├── SuccessCases.jsx
│   │   ├── Methodology.jsx
│   │   └── Footer.jsx
│   ├── data/
│   │   └── content.js
│   ├── App.jsx
│   ├── main.jsx
│   └── index.css
├── public/
├── index.html
├── package.json
├── vite.config.js
├── tailwind.config.js
└── postcss.config.js
```

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Build para Producción

```bash
npm run build
```

## Tecnologías

- React 18
- Vite
- Tailwind CSS
- Lucide React (iconos)

## Características

- Diseño responsive y moderno
- Navegación suave entre secciones
- Menú móvil lateral
- Optimizado para SEO
- Componentes modulares y reutilizables
